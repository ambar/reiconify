# How to Align

## RECOMMENDED

import {
  CenterDemo,
  FlexContainerDemo,
  VerticalAlignMiddleDemo,
  TopOffsetHackDemo,
} from './Align'

### Use strut hack with a `inline-flex` wrapper, simply and self contained, see also [react-inline-center](https://www.npmjs.com/package/react-inline-center).

```js
<button>
  <span style={{display: 'inline-flex', alignItems: 'center'}}>
    {'\u200b'/* ZWSP(zero-width space) */}
    <Icons.Heart text />
  </span>
  Like
</button>

// This is how `center` prop works:
<button>
  <Icons.Heart text center />
  Like
</button>
```

<CenterDemo />

### Use `flex` or `inline-flex` container.

```js
<button style={{display: 'inline-flex', alignItems: 'center'}}>
  <Icons.Heart text />
  Like
</button>
```

<FlexContainerDemo />

## NOT RECOMMENDED

### Use `vertical-align: middle`, requires extra style and tag for text.

```js
<button>
  <Icons.Heart text style={{verticalAlign: 'middle'}} />
  <span style={{verticalAlign: 'middle'}}>Like</span>
</button>
```

<VerticalAlignMiddleDemo />

### Use `top` offset hack, it's font-specific.

```js
<button>
  <Icons.Heart
    text
    style={{verticalAlign: 'middle', position: 'relative', top: '-.1em'}}
  />
</button>

// top = (xHeight/2 - capHeight/2) / unitsPerEm â‰ˆ .5em/2 - .7em/2
// query font metrics: https://opentype.js.org/font-inspector.html
```
